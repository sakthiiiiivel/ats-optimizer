# ATS Optimizer 🚀

An AI-powered system to analyze resumes against job descriptions and improve ATS compatibility.

## 🔥 Features
- Resume parsing (PDF)
- Job description analysis
- Skill extraction
- Resume vs JD matching
- Missing skills detection

## 🧠 Tech Stack
- Python
- FastAPI
- NLP
- Machine Learning (planned)
